Mostly I followed this tutorial to set up giscus, upto the configuration of giscus parameters in the hugo configuration file. There instead I set the following in the [params] section (replace "repo_id" etc. by the correct values):

[params]
  # Other parameters
  ...
  # Giscus paramaters
  giscus_commentable = {page = true, post = true}
  giscus_repo = "repo_name"
  giscus_repo_id = "repo_id" 
  giscus_category = "Announcements"
  giscus_category_id = "category_id"
  giscus_mapping = "pathname"
  giscus_label = "comment"
  giscus_theme = "preferred_color_scheme"
  giscus_reactions_enabled = 0

These parameters are read by a giscus.html file stored in the layouts/partials/ of the theme containing:

<script src="https://giscus.app/client.js"
        data-repo="{{site.Params.giscus_repo}}"
        data-repo-id="{{site.Params.giscus_repo_id}}"
        data-category="{{site.Params.giscus_category}}"
        data-category-id="{{site.Params.giscus_category_id}}"
        data-mapping="{{site.Params.giscus_mapping}}"
        data-reactions-enabled="{{site.Params.giscus_reactions_enabled}}"
        data-theme="{{site.Params.giscus_theme}}"
        crossorigin="anonymous"
        async>
</script>
<noscript>Please enable JavaScript to view the comments powered by giscus.</noscript>

Finally I added the following line at the bottom of the file layouts/partials/footer.html:

{{ if .Params.giscus }}{{ partial "giscus.html" . }}{{ end }}

Now if you set giscus: true in the header of a post (similar to the MathJax solution above), comments should appear on that post.
